# Workplan

## Projekt: lpai - Lineær Programmering App

## Mål
Interaktiv web-app til lineær programmering med klassiske eksempler og problem-formulator.

## Features

### Fase 1 ✅
- [x] Vite + React + TypeScript setup
- [x] 5 klassiske eksempler med naturligt sprog + formel repræsentation
  - Diætproblemet
  - Produktionsplanlægning
  - Transportproblemet
  - Blandingsproblemet
  - Vagtilrettelægning
- [x] Teori-sektion med LP-begreber

### Fase 2 🔲
- [ ] Simplex solver (funktionsdygtig, ikke kun preset løsninger)
- [ ] Problem-formulator: AI hjælper brugeren med at oversætte naturligt sprog til ligninger
- [ ] Interaktive målfunktioner (vælg max/min, se forskellige resultater)

### Fase 3 🔲
- [ ] Visualisering af løsningsrum
- [ ] Exportér resultater
- [ ] Flere eksempler fra litteraturen

## Teknisk
- Frontend: Vite + React + TypeScript
- Solver: TODO (Overvej simplex-js eller selv-impl)
- Styling: Plain CSS (minimal afhængigheder)
